# Colors of Goodbye

Data narrative exploring how sunsets shift in colour and light over time—and how those shifts translate across planetary atmospheres.

Static site: HTML, CSS, and JavaScript.

## Authors

- Kapadne Siddhant
- Jaisriram Rajkumar

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.
